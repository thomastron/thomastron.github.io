~thomastron~

 

github.com/thomastron/

This collection of Jupyter notebooks is primarily equations and code snippets, but it includes explanations on using the notebooks and understanding python. You can copy the files from the Github repository and use the files locally by hosting a Jupyter server on your local machine. Another option is to use the free web-hosted service Binder that creates and hosts a Jupyter server based on the github repo. This doesn't work on mobile devices:
